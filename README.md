# ManRise - A Full Stack E-Commerce Website

## Goal

- Built a full-stack e-commerce application with `Next.js 15`, `TypeScript`, `TailwindCSS v4`, `Prisma`, and `PostgreSQL`.
- Implemented `secure authentication and role-based dashboards` for admin and users.
- Applied `mobile-first responsive design`, `DRY`, `modular design`, and `clean architecture principles` for scalability.
- Practiced `Test-Driven Development (TDD)` using `Vitest` & `React Testing Library` to ensure reliability.
- Optimized performance with `efficient Prisma queries, lazy loading, and caching strategies`.
- Integrated `ESLint, Prettier, and Tailwind Prettier plugin` for consistent code style and quality.
- Configured `CI/CD pipelines with GitHub Actions` for automated linting, testing, and deployments.
- Deployed frontend on `Vercel` and backend/database on `cloud platforms (Railway/PlanetScale)`.
